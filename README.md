Đây là bài tập mã hoá và giải mã chữ ký số RSA, được viết bằng ngôn ngữ JavaScript và thư viện Reactjs
Cách chạy dự án:
b1: clone dự án
gitclone https://github.com/Nguyenthithuhang0406/RSA-ATTT.git
b2: cài đặt lại package
npm install
b3: chạy lệnh 
npm run dev
